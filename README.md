# App de Control de Pendientes

Aplicación web responsiva para gestionar pendientes personales, compuesta por:

- Backend en **Python FastAPI** con base de datos **SQLite**.
- Frontend en **HTML, CSS y JavaScript**.
- Uso de colores cálidos y soporte para:
  - Fecha de creación
  - Descripción
  - Importancia (alta, media, baja)
  - Estado (pendiente o completado)
  - Fecha de atención (cuando se marca como completado)

Este proyecto fue **generado con Perplexity AI** a partir del siguiente prompt (instrucción del usuario):

> Crea una web responsiva que tenga tanto el frontend (HTML, css, Javascript) y el backend (python fastapi con base de datos sqlite) para llevar el control de mis pendientes. Para cada pendiente considera fecha creación, descripción, importancia (alta, media, baja), estado (pendiente o completado) y fecha de atención en caso lo marque como completado. Utiliza colores cálidos.

---

## Tecnologías utilizadas

- Python 3.x  
- FastAPI  
- Uvicorn (servidor ASGI para desarrollo)  
- SQLite (base de datos embebida)  
- HTML5, CSS3 y JavaScript
